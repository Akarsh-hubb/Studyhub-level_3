# 🚀 StudyHub - Performance Optimization (Level 3 Task 2)

A Django-based StudyHub application optimized for performance as part of the **Codveda Technologies Web Development Internship (Level 3 - Task 2)**.

The project focuses on improving website performance, reducing page load times, optimizing static assets, and enhancing the overall user experience using Google Lighthouse.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Akarsh-hubb/Studyhub-level_3.git
```

Navigate into the project

```bash
cd Studyhub-level_3/studyhub-task_1
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the server

```bash
python manage.py runserver
```

Visit

```
http://127.0.0.1:8000/
```
